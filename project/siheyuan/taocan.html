  • 现代简约风格MODERN STYLE

    将设计元素、色彩、照明、材料简化到极致，运用黑、白、灰展现特有的明快及冷调，简约并实用，体现出工业化社会生活的精致与个性，满足现代人的生活品味。

  • 后现代风格POSTMODERN STYLE

    主张探索创新造型，讲究人生百味，采用非传统的混合、叠加、错位、裂变等手法，象征或隐喻一种融感性与理性、集传统与现代于一体的室内风格

  • 欧式风格EUROPEAN STYLE

    又称西洋古典风格，强调以华丽的装饰、浓烈的色彩、精美的造型达到雍容华贵的装饰效果，带来无限的舒服触感、惬意浪漫，达到整体和谐的最高境界。

  • 地中海风格MEDITERRANEAN STYLE

    充分利用每一寸空间，集装饰、应用于一体，特有罗马柱般的装饰线简洁明快、大方自然，用现代工艺呈现别有情趣的格调，散发着蔚蓝色的浪漫情怀。

  • 美式田园风格AMERICAN PASTORAL STYLE

    源自美国乡村的随意，摒弃过多的奢华，兼具田园的优美造型和功能配备，简洁明快、温暖舒适，造就自在、随意的生活方式，不经意中成就另外一种休闲式的浪漫。

  • 北美风格NORTH AMERICA STYLE

    注重建筑细节，既有古典情怀，又具外观简洁大方，融多种风情于一体的多元化风格，反映出以自然为依托，以人文本的现代设计理念。

  • 中式风格CHINESE STYLE

    继承唐代、明清时期宫廷建筑的精华，将经典元素提炼加以丰富，以木为主，搭配简洁、硬朗的线条，气势恢宏、壮丽华贵，彰显中式家居追求内敛、质朴的设计风格。

  • 北欧风格NORDIC STYLE

    以简洁著称，原始石材面及木纹暴露于室内，但主题又偏向于现代结构，注重流畅的线条设计，反映出现代都市人进入新时代的某种独特取向与旋律。

  • 田园风格POSTORAL STYLE

    运用西欧设计的大气，小清新格子碎花的脱俗，配合浪漫高贵的表现手法，倡导回归自然，在当今高科技、快节奏的社会生活中获取一丝悠闲、舒畅、自然的生活情趣。
